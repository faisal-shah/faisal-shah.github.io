Sabeel Class Recordings

Audio recordings of Sabeel Institute classes. Students listen and keep up; teachers track who has caught up. Students sign in with the email their teacher registered; staff sign in with their Sabeel Google account and are approved by an administrator.

Use it in a browser

Open the web app Works on any device. Nothing to install.

Install on Android

Current build: v0.1.3. The download link stays the same when a new build is published, so this page is always the latest.

Download for Android (39 MB) For any phone from roughly 2016 onward. Start here. Download the all-devices build (93 MB) Only if the one above won't install — it's larger but runs anywhere.

User manual (PDF) Every screen, on a phone and on a computer, with step-by-step guides.
