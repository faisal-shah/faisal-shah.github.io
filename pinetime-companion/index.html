PineTime Companion

Manage your PineTime smartwatch over Bluetooth — schedules and reminders, alarms, prayer times, weather, step history, phone-notification and music forwarding, firmware updates, and Apple Find My beacon keys.

Use it in a browser

Open the web app Chrome or Edge on desktop (needs Web Bluetooth). Nothing to install.

The browser version manages watches and beacon keys. Firmware updates, notification & music forwarding, and the Find My map need the Android app below.

Install on Android

The download link stays the same when a new build is published, so this page always points at the latest.

Download for Android (~38 MB) The full app, for any 64-bit phone (roughly 2016 onward). Start here.

Android will warn about installing outside the Play Store and ask you to allow it for your browser. That is expected for an app distributed directly like this.

Other Android builds and the desktop apps (Windows, macOS, Linux) are on the releases page.

There is no iPhone version, and the web app needs a Chromium browser (Chrome or Edge) — Safari and Firefox don’t support Web Bluetooth.
